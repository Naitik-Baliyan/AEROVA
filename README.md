## 🚀 The Vision
Most environmental applications stop at delivery, overloading users with anxieties regarding poor air metrics without giving them active coping mechanisms. 

**Aerova** is engineered as an **"Action Engine."** It blends high-fidelity, live geospatial telemetry from physical monitoring stations with active, zero-latency pulmonary meditation to transform passive awareness into real, immediate physical restoration.

---

## ✨ Key Features
*   **📍 Real-Time Physical Telemetry**: Direct spatial routing to the official **World Air Quality Index (WAQI) Physical Government Sensors**, delivering certified local station data with 100% accuracy.
*   **🧘 BreatheNow Sanctuary**: An immersive, offline-capable meditation suite. Features high-fluidity, mathematically tuned vector expansion rings paired with slow-pitch guided audio cues.
*   **☁️ Secure Cloud Architecture**: End-to-end user registration, secure encrypted authentication, and personalized profile synchronization via **Supabase Postgres**.
*   **🎨 Glassmorphic Luxury UI**: Premium design language featuring vibrant gradients, custom-weighted typography, and responsive system-clock greeting matrices.
*   **🛡️ Resilient Sensor Defense**: Robust fallback architecture guarding against offline physical stations or missing specialized atmospheric gas channels.

---

## 🛠️ Technical Architecture
*   **Frontend Mobile**: React Native & Expo SDK 51+
*   **Backend SaaS**: Supabase (Auth & PostgreSQL Database)
*   **Audio Synthesis**: `expo-av` high-performance sound buffering and dynamic OS-level Speech synthesis.
*   **Vector Graphics**: `react-native-svg` for responsive performance-tuned rendering of physical telemetry dials.
*   **Pipeline**: Configured with EAS Cloud for streamlined, asset-sanitized standalone Android APK generation.

## 📥 Getting Started & Installation

### Prerequisites
Ensure you have Node.js (v18+) and the Expo CLI tools installed.

### 1. Clone the Repository
```bash
git clone https://github.com/Naitik-Baliyan/AEROVA.git
cd AEROVA
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Native Development Sandbox
```bash
npx expo start
```
*Scan the QR code inside your terminal with the Expo Go app on your Android or iOS device to launch the live sandbox instantly!*
